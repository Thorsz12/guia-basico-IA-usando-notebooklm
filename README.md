## Mini Guia de Estudos 
### Resumo Gerado Pela IA e Revisado 
A inteligência artificial (IA) é um campo da ciência da computação que desenvolve sistemas e algoritmos capazes de simular o raciocínio humano para realizar tarefas de forma autônoma. Seu funcionamento básico depende do processamento de grandes volumes de dados combinados com algoritmos, que permitem ao software identificar padrões e tomar decisões inteligentes.

Os principais métodos de aprendizado são o machine learning (aprendizado de máquina), onde a IA melhora seu desempenho com base em dados históricos, e o deep learning (aprendizado profundo), que utiliza redes neurais inspiradas no cérebro humano para processar informações altamente complexas. Uma aplicação moderna e central nesse campo são as LLMs (Large Language Models), modelos de linguagem treinados com bases massivas de dados textuais para compreender e gerar linguagem humana de forma natural e fluida, servindo de base para ferramentas como o ChatGPT.

Atualmente, as tecnologias disponíveis pertencem à categoria de IA Limitada ou Estreita, pois são projetadas para funções específicas, como assistentes de voz e reconhecimento facial, diferindo da IA Geral, que ainda é teórica e teria capacidades intelectuais similares às humanas. Embora a IA proporcione maior produtividade e avanços significativos em áreas como a medicina, seu uso também gera debates cruciais sobre ética, privacidade e o risco de desemprego estrutural devido à automação.
### Principáis Conceitos
* **Inteligência Artificial (IA)** É a tecnologia que permite que máquinas e programas imitem o comportamento e o raciocínio humano para realizar tarefas de forma autônoma.
* **Machine Learning** É um método onde a IA aprende a identificar padrões sozinha com base em grandes volumes de dados e experiências passadas
* **LLMs**São IAs treinadas com bases gigantescas de texto para entender, conversar e gerar respostas de forma natural, como o ChatGPT.
* **IA Generativa** É o tipo de IA focado em criar conteúdos novos do zero, como textos, imagens, músicas e vídeos, a partir de comandos simples
* **Deep Learning** É uma evolução do aprendizado de máquina que utiliza redes neurais (inspiradas no cérebro humano) para resolver problemas muito complexos, como reconhecer imagens ou escrever textos
